# SleepAuality
Aplikasi ini bertujuan untuk membantu pengguna mengevaluasi kualitas tidur secara sederhana. Pengguna menginput jam tidur, jam bangun, serta kategori umur, Maka aplikasi akan menghitung dan memberikan penilaian apakah kualitas tidur kiita sudah cukup atau masih belum, berdasarkan kebutuhan tidur sesuai kelompok usia. Dengan itu pengguna lebih sadar terhadap pola diturnya sehari-hari apakah sudah ideal atau belum.
